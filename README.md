# 🌍 TravelPlatform-FullStack

Projeto Full Stack de plataforma de viagens desenvolvido durante minha formação Full Stack na Recode Pro.

O projeto evoluiu desde páginas estáticas em HTML/CSS até uma arquitetura moderna utilizando React no front-end e C# no backend.

---

# 🚀 Tecnologias Utilizadas

## Front-End
- HTML5
- CSS3
- JavaScript
- Bootstrap
- React

## Back-End
- C#
- .NET
- API REST

## Ferramentas
- Git
- GitHub
- VS Code
